## 🚀 核心亮點專案：AI 開發流程自動化 (AI-Powered SDD Workflow)
**角色：專案負責人 (PM / SA / Developer)**

主導並實作一套端到端的 AI 開發自動化工具鏈，旨在實現「規格驅動開發 (SDD)」並極大化團隊產能。

AI 驅動開發 (LLM-Ops)： 實作企業級 AI 自動化流程，透過 CLI 工具整合 GitHub SDK 與 Prompt Validation，達成從需求分析到程式產出的自動化閉環，旨在實現「規格驅動開發 (SDD)」並極大化團隊產能。
* **自動化需求分析：**
    - 整合 **STT (Speech-to-Text)** 進行訪談紀錄。
    - 指導 AI 進行需求分析，並自動生成高品質的 **Markdown 技術文件** 與系統規格書。
* **自研開發 CLI 工具：**
    - 實作統一開發環境的 CLI。
    - 內建 **SSH TUI** 介面與 **Prompt Validation**，確保開發者能有效控管 AI 產出。
* **人機協作流程 (HITL Mechanism)：**
    - **Issue 階段：** 透過 CLI 提出 Issue，自動進行模板與 Prompt 驗證。
    - **開發階段：** 整合 GitHub SDK 自動化初始化 DevEnv、Compile 與 Deploy。
    - **審核階段：** AI 預審 Code Review (HITL)，確保代碼符合團隊風格，工程師僅需專注於最終決策與高價值測試。
